# VillagerPro 2.0

沉浸式村庄经营生态系统 - Minecraft Spigot/Paper 插件

## 功能特性

- 村庄系统：创建和管理你的村庄
- 村民招募：招募农民、渔夫、牧羊人等职业村民
- 村民升级：提升村民技能和效率
- 村庄仓库：存储村民产出的物品
- 访客系统：商人、旅行者、节日使者
- 联盟系统：与其他玩家合作

## 依赖

- Spigot/Paper 1.19.4+
- Vault
- PlayerPoints (可选)
- ItemsAdder (可选)

## 构建

```bash
gradle build
```

## 安装

1. 将生成的 `VillagerPro-2.0.0.jar` 放入服务器的 `plugins` 目录
2. 重启服务器
3. 根据需要修改 `config.yml`

## 使用

- `/village` - 打开村庄管理界面
- `/villager` - 村民管理命令
- `/upgrade` - 升级命令
